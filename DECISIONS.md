## 2026-05-18: Редизайн попапа карточки квартиры (по образцу auto.ru)

**Что:** полная переработка визуала карточки квартиры. Все изменения — в restructuring-блоке (setTimeout внутри openAptCard) + CSS.

**Островная вёрстка:**

* Фон оверлея `#F2F2F7` (серый), контент — белые острова с `border-radius:14px`
* Top-island: `border-radius:0 0 14px 14px`, `padding:0`, `overflow:visible`, `z-index:2`
* Остальные острова: `padding:16px`, `margin-bottom:4px`
* Step 11 группирует смежные блоки в острова, пропускает скрытые (`offsetHeight===0`)

**Ценовая строка (price-line):**

* Строка 1: цена + бейдж скидки (-15%) + \[В брони] + \[Hoff] — всё в одном flex-row
* Строка 2: зачёркнутая старая цена (отдельным блоком, margin-top:4px)
* Строка 3: подзаголовок (формат · площадь · район) + бейдж «Видовая квартира»
* «Под запрос» — отображается тем же шрифтом что обычная цена (fallback в step 4)
* Hoff: убран текст «50 000 ₽», оставлена только лого + тултип по клику
* **Критический баг**: id `#apc-price` нельзя переименовывать (ломает повторное открытие). Стилизация через класс `apc-price-main-styled`.

**Бейдж «Видовая квартира»:**

* Фиксированный текст «Видовая квартира» + иконка глаза
* По клику — тултип с деталями (напр. «Вид на Малую Невку»)
* Если значение уже начинается с «Вид на» — оставляется как есть
* Иначе — «Вид на » + значение без изменения регистра (имена собственные)
* Тултип позиционируется с учётом краёв экрана (как bookedTip)

**Остров характеристик:**

* 4 ячейки в одну flex-строку: кухня | этаж | срок сдачи | отделка
* Кухня всегда показывается (прочерк «—» у студий)
* Ячейки: `flex:1 1 0`, first-child left, last-child right, средние center
* Внутри ячейки: `.apc-spec-inner` (inline-block, text-align:center) — подпись центрируется под значением
* Разделители: `::after` pseudo-элемент, 1px, #E5E5EA
* **TODO:** добавить иконки (SVG) к каждой характеристике (по образцу auto.ru)

**Кнопка «Получить консультацию»:**

* Цвет: `#00a73e` (зелёный, как на auto.ru)
* Уменьшена: padding 18→14px, font 16→15px, radius 14→12px
* Аватар Елены полностью удалён (HTML, CSS, IntersectionObserver)

**Прочее:**

* Стрелка в блоке ремонта: `polyline 9,6→15,12→9,18` (вправо), `rotate(90deg)` при открытии (вниз)
* Тултипы не обрезаются: top-island `overflow:visible`, `z-index:2`
* view-row скрывается в step 2 (вид показывается бейджем)
* Расстояния: plan→gallery = gallery→price = 18px

**Почему:** улучшение UX карточки, ориентир — мобильная карточка auto.ru. Компактнее, информативнее, кнопка заметнее.

**Цена:** \~20 правок в CSS + JS restructuring. Код усложнился (step 0 cleanup стал объёмнее из-за idempotent-логики).
